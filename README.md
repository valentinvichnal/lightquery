# lightquery
Lightweight dom manipulation library (IE9+) - **3kb only**
```
npm i --save lightquery
```

## Usage:
Include before the closing body tag:
```
<script src="lightquery.js"></script>
```
or require it from the npm module
```
var $ = require('lightquery');
```
